# BymseRead

Online reader for PDF books

# Local dev setup

1. Install https://ghostscript.com/releases/gsdnld.html
2. Add http://read.bymse.local:5173/ to `chrome://flags/#unsafely-treat-insecure-origin-as-secure`
3. Frontend address: http://read.bymse.local:5173/
3. Default keycloack user: default:default
